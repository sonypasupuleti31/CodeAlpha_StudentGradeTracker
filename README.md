This is a Java-based console application that allows users to:

Add student names and grades  
Calculate average, highest, and lowest grades  
View summary reports  
Store data using arrays or ArrayLists  
Console-based interaction using switch-case

## 📁 Structure
- `StudentGradeTracker.java`: Main Java source file

## 🚀 How to Run
1. Compile: `javac StudentGradeTracker.java`
2. Run: `java StudentGradeTracker`
